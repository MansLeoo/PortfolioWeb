TerrariskVR

Application de sensibilisation aux risques d'éboulements pour GRDF. Simulation physique temps réel et pédagogie immersive.

Voir le code Lire le rapport

Contexte & Objectifs

L'objectif de ce projet était de créer un outil de formation immersif pour les techniciens GRDF. Les méthodes d'apprentissage classiques (PowerPoint) ne permettaient pas de visualiser correctement le danger d'un éboulement de terrain lors de travaux de fouille.

Nous avons donc conçu un scénario en Réalité Virtuelle où l'utilisateur doit sécuriser une zone avant qu'un accident simulé par le moteur physique ne se produise.

Challenges Techniques

1. Simulation Physique Optimisée

Pour simuler l'éboulement de manière réaliste sur un casque autonome (Quest 2), j'ai dû éviter d'utiliser trop de `Rigidbodies`. J'ai opté pour une approche hybride : une animation pré-calculée (Vertex Animation) pour la masse de terre, couplée à quelques objets physiques pour l'interaction avec le joueur.

2. Interaction Main-Objet (Hand Tracking)

Utilisation du SDK Oculus Interaction pour permettre la manipulation des outils (pelles, barrières) sans manettes, directement avec le suivi des mains, pour plus d'immersion.
